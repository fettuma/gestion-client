# ClientBase — Système de Gestion des Clients

Application web PHP/MySQL développée durant un stage chez **Ultra Soft SARL**.

## Fonctionnalités
- Gestion des clients
- Commandes & détails commandes
- Produits & catégories
- Factures avec tableau de bord

## Technologies
PHP 8 · MySQL · HTML5/CSS3 · JavaScript

## Installation
1. Cloner le repo
2. Copier `config.php.example` → `config.php`
3. Importer `gestion_client.sql`
4. Lancer XAMPP
